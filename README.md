# Monokai Dark Modern

A VS Code theme that brings together the best of both worlds: **Monokai's** iconic, vibrant syntax highlighting with **Dark Modern's** clean, minimal UI.

## Features

- 🎨 **Monokai Syntax Colors** - The beloved pink, green, yellow, purple, cyan, and orange palette
- 🖥️ **Dark Modern UI** - Clean, minimal interface with subtle blue accents
- ✨ **Semantic Highlighting** - Enhanced language-aware syntax coloring

## Credits

This theme is a combination of two official VS Code themes:

- [Monokai](https://github.com/microsoft/vscode/tree/main/extensions/theme-monokai) - Syntax highlighting
- [Dark Modern](https://github.com/microsoft/vscode/tree/main/extensions/theme-defaults) - UI styling
